## Project Description

This project is a Java-based Student Attendance Management System designed to help administrators and teachers manage student attendance records efficiently. The system uses a MySQL/MariaDB database for data storage and provides a simple graphical user interface for daily operations.

---

## Features

* Secure user login (Admin / Teacher)
* Add, update, and delete students
* Mark student attendance
* View attendance records and reports
* Database connectivity using JDBC

---

## Technologies Used

* Java (Swing)
* MySQL / MariaDB
* JDBC
* Git & GitHub

---

## Database Setup

1. Create a database in MySQL/MariaDB:

   ```sql
   CREATE DATABASE student_attendance;
   ```
2. Import the SQL file located in the project:

   ```
   /database/database.sql
   ```
3. Update database credentials in the Java DB connection file if required.

---

## How to Run the Application

1. Clone the repository from GitHub.
2. Open the project in NetBeans or IntelliJ IDEA.
3. Ensure MySQL/MariaDB is running.
4. Import the database using the provided SQL file.
5. Configure database username and password in the project.
6. Run the `Main.java` file.
